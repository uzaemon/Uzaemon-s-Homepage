! Read before you start !

  • Examples in library "IPTULIB" and directory "/iptu" are protected by copyright. You need written permission to redistribute them.

  • You may redistribute "Utilities" under the terms and conditions included in the each distribution.

  • All examples and utilities should run fine on OS/400 V5R2, which had been a main test bed.

    • Though this CD-ROM contains executables partially, you should compile (make) executables by yourself. Obviously, you need ILE or Java compiler to use the utilities. In addition, IBM supplied library QSYSINC is required in many cases.
    • As for version compatibility of examples in IPTULIB, read corresponding part of the book. Basically, you should be able to run most examples on V4R3 and later with Original HTTP Server, and V4R5 and later with Apache HTTP Server (assuming that appropriate level of PTFs are applied).
    • Most examples and utilities will not run on V3R2 and earlier (A.K.A. CISC machines). The V3R2 save files are provided mainly for V3R7/V4R1/V4R2 users.
    • Analog and wcount may work on V4R2 or later. The zlib, TIFF2PDF, SPL2STMF, SNDM and CRTCSVF may work on V3R7 and later.
    • RCVM may work with JVM 1.1.8 or later if you don't use UTF-16LE. JVM 1.3 and OS/400 V5R1 and later are recommended.

  • For updates of the book, access to "Internet Programming, Tools, and Utilities" page.
